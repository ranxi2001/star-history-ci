# Contributing

[中文](#中文)

Issues and pull requests are welcome. Please check existing issues before opening a new one.

Keep pull requests focused. One PR should solve one problem or add one feature.

## Guidelines

- Use clear titles and include relevant workflow logs for bug reports.
- Update README files and examples when behavior changes.
- Keep generated files out of the default branch unless they are required source files.
- Prefer small, reviewable changes.

## 中文

欢迎 PR 和 Issue。内容补充、错误修正、新模块建议均可。

提 Issue 前请先检查是否已有相关讨论。PR 建议一个 PR 只做一件事。

## 规范

- Bug 反馈请使用清晰标题，并附上相关 workflow 日志。
- 行为变化时同步更新 README 和 examples。
- 除非是必要源码，不要把生成文件提交到默认分支。
- 优先提交小而清晰、方便 review 的改动。
